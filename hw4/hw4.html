Problem 3: DAC Output (Right Aligned)

A 12-bit DAC with Vref=3V reads 0x08EA (right-aligned). The animation converts the rightmost 12 bits (2282) to a voltage of 1.671V.
Bits

0x08EA (right 12 bits) = 2282

Vout = 3 * 2282 / (2^12 - 1)

Vout = 1.671 V

Problem 5: 1-bit DAC Maximum Error

In a 1-bit DAC, if R value can vary +/-10%, what is the maximum error on Vout? The animation highlights 0.05 Vref.
Options

R1 = 1.1, R2 = 0.9

Vout = Vref * 0.9 / (1.1 + 0.9) = 0.45 Vref

Error = |0.45 Vref - 0.5 Vref| = 0.05 Vref

Problem 8: ADC Maximum Quantization Error

If Vref = 6V and ADC resolution is 12 bits, what is the max quantization error? The animation highlights 0.73 mV.
Options

LSB = 6 / (2^12 - 1)

Max Error = 1/2 * LSB

Max Error = 0.73 mV
